FAQs

Basics

What is Learn Token?

Based on Ethereum technology, Learn Token is a utility that can be used as an account unit between between sponsors, advertisers, students in a new blockchain-based learning platform. Note that the token is not a currency.

How can I buy Tokens?

Tokens can be bought with Smart Contracts. The information regarding Token purchases will be provided soon.

Are Tokens transferable? Where are Tokens traded?

The purpose of Learn Tokes is for it to be used on the Learn.xyz platform as an exchange unit. Yes, Tokens are transferable. The information regarding places where Tokens can be traded will be provided soon.

What do Tokens represent?

Tokens represent a utility used in order to use the Learn.xyz platform - a new blockchain technology-based learning platform. Token ownership means only the right to use them in order to obtain services from the Learn.xyz platform as well as enable the usage and interaction with the platform. Token ownership does not represent any stakes, shares, rights to future profits or any ownership of any intellectual property or any other form of participation related to the Learn.xyz platform.

Where will Tokens be used?

The Token will be used on the Learn.xyz platform in order to obtain services from the platform and conduct translations on the platform. More information will be provided soon.

Which wallets can be used to store Token?

Learn Token is an ERC20 token. In order to store the Learn Token(s) any wallet that is compatible with ERC20 can be used.

Will I lose my Token after 6 months?

No, you will not use any Token after 6 month period, as the Learn tokes in built according to ERC20 token standards.

What is the roadmap?

The Learn.xyz roadmap can be found here.

I transferred some Token from one wallet to another and it never arrived. Can you help me?

We do not store any keep or store any information regarding wallet usage and unfortunately cannot help you directly. The best way in order to solve such problem is contacting the providers of the wallet that you are using.
